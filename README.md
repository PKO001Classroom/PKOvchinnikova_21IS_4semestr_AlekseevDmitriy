## Описание

Система управления учебным процессом с автоматическим выставлением оценок на основе индикаторов освоения ФГОС СПО (Федеральных государственных образовательных стандартов среднего профессионального образования). 

Desktop-приложение для автоматизации выставления, хранения и просмотра оценок с ролевым разделением (преподаватель/студент) и строгим соответствием требованиям ФГОС СПО.

## Особенности системы

1. **Оценка по ФГОС**: Оценки выставляются не напрямую, а через выбор освоенных индикаторов
2. **Автоматический расчет**: Система автоматически рассчитывает оценку и процент освоения
3. **Детальная аналитика**: Подробная информация о каждом индикаторе освоения
4. **Цветовая индикация**: Визуальное отображение уровня освоения
5. **Ролевой доступ**: Разделение прав между преподавателями и студентами
6. **Локальное хранение**: Все данные хранятся локально в SQLite базе

## Структура оценки по ФГОС

### Компетенции:
- **ПК** - Профессиональные компетенции
- **ОПК** - Общепрофессиональные компетенции  
- **УК** - Универсальные компетенции

### Индикаторы освоения:
Для каждой компетенции определены индикаторы - конкретные умения и навыки, которые должен освоить студент.

### Шкала оценок:
| Балл | Процент освоения | Уровень |
|------|------------------|---------|
| 5    | 86-100%          | Высокий уровень |
| 4    | 67-85%           | Повышенный уровень |
| 3    | 48-66%           | Базовый уровень |
| 2    | 0-47%            | Не сформировано |

## Установка и запуск

### Локальный запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/PKO001Classroom/PKOvchinnikova_21IS_4semestr_AlekseevDmitriy
   cd PKOvchinnikova_21IS_4semestr_AlekseevDmitriy/edu_journal
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Запустите приложение:
   ```bash
   python main.py
   ```

### Запуск через Docker

1. Соберите Docker образ:
   ```bash
   docker build -t edu-journal .
   ```

2. Запустите приложение в контейнере:
   ```bash
   docker-compose up
   ```

3. Или используйте production конфигурацию:
   ```bash
   docker-compose -f docker-compose.prod.yml up -d
   ```

### Ключевые функции:

**Расчет оценки по ФГОС:**
```python
def calculate_grade(percentage):
    """Преобразует процент в оценку по ФГОС"""
    if 86 <= percentage <= 100:
        return 5, "высокий уровень"
    elif 67 <= percentage <= 85:
        return 4, "повышенный уровень"
    elif 48 <= percentage <= 66:
        return 3, "базовый уровень"
    else:
        return 2, "не сформировано"
```

**Валидация комментария:**
```python
def validate_comment(comment, min_length=10):
    """Проверяет комментарий на соответствие требованиям"""
    if len(comment) < min_length:
        return False, f"Комментарий должен быть не менее {min_length} символов"
    
    if 'ПК' not in comment and 'ОПК' not in comment:
        return False, "Рекомендуется указать код компетенции (ПК или ОПК)"
    
    return True, "OK"
```

## Разработка

### Технологический стек:
- **Backend**: Python 3.11+
- **GUI**: PyQt5 5.15.9
- **База данных**: SQLite3
- **Контейнеризация**: Docker + Docker Compose

### Архитектура:
- **Паттерн**: Упрощенная MVC (Model-View-Controller)
- **Хранение данных**: Локальная SQLite база
- **Интерфейс**: Desktop приложение на PyQt5

### Этапы разработки:
1. **Неделя 1**: Базовая структура проекта + SQLite база данных
2. **Неделя 2**: Система авторизации и роли
3. **Неделя 3**: Окно студента (просмотр оценок)
4. **Неделя 4**: Окно преподавателя (выставление оценок)
5. **Неделя 5**: Валидации по ФГОС + бизнес-логика
6. **Неделя 6**: Тестирование и отладка
7. **Неделя 7**: Упаковка и документация
8. **Неделя 8**: Деплой и финальные правки

## Особенности реализации

1. **Локальное хранение**: Все данные хранятся на компьютере пользователя
2. **Простая аутентификация**: Без шифрования паролей (для демо-версии)
3. **Минимальные зависимости**: Только PyQt5 и Python стандартная библиотека
4. **Кроссплатформенность**: Работает на Windows, Linux, macOS
5. **Простота развертывания**: Один .exe файл или pip пакет
