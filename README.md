# Учебный журнал с учетом ФГОС СПО

Система управления учебным процессом с автоматическим выставлением оценок на основе индикаторов освоения ФГОС.

## Особенности системы

1. **Оценка по ФГОС**: Оценки выставляются не напрямую, а через выбор освоенных индикаторов
2. **Автоматический расчет**: Система автоматически рассчитывает оценку и процент освоения
3. **Детальная аналитика**: Подробная информация о каждом индикаторе освоения
4. **Цветовая индикация**: Визуальное отображение уровня освоения

## Структура оценки по ФГОС

### Компетенции:
- **ПК** - Профессиональные компетенции
- **ОПК** - Общепрофессиональные компетенции  
- **УК** - Универсальные компетенции

### Индикаторы освоения:
Для каждой компетенции определены индикаторы - конкретные умения и навыки, которые должен освоить студент.

### Шкала оценок:
- **5 (86-100%)**: Высокий уровень - полное освоение
- **4 (67-85%)**: Повышенный уровень - уверенное применение
- **3 (48-66%)**: Базовый уровень - освоение с поддержкой
- **2 (0-47%)**: Не сформировано - требуется обучение
